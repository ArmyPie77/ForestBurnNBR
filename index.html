  • API runs can take several minutes.

Landsat

30 m multispectral imagery with deep archives and frequent revisits—ideal for before/after fire views.

dNBR

dNBR compares pre- and post-fire NBR to reveal burn severity. Values above ~0.27 often flag notable change.

Preloaded tiles

Preloaded burn tiles cover key seasons across Quebec and load quickly as georeferenced PNG overlays. Use the sidebar toggles to explore each season without waiting for processing.

API analysis

The API pulls surface reflectance scenes for your path/row and dates, computes dNBR, and serves a web overlay. Wider date ranges help dodge clouds.
